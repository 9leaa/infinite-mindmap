# 无限画布思维导图 V15.7

## 本版修复

### 双悬浮工具折叠状态

- 蓝色 `◇ 导图`：思维导图悬浮工具。
- 琥珀色 `✎ 画笔`：绘图悬浮工具。

### Pencil 与手指

- 识别 `touchType=stylus`。
- 结合接触面、压力和最近笔尖位置过滤 Pencil 产生的 Touch 事件。
- 只有真正的手指拖动超过阈值后才移动画布。
- 无限画布和 PDF 阅读区使用同一套识别逻辑。

### PDF 性能

- 同时最多渲染两页。
- 首屏页面优先。
- 缩小预加载范围。
- PDF 渲染像素比最高约 1.45。
- 屏幕外页面使用 `content-visibility` 减少布局与绘制。
- PDF 数据的 Base64 解码结果在当前会话缓存。

### PDF 阅读界面

- 占满整个应用窗口。
- 适应宽度。
- 常驻滚动条。
- 竖向页码进度条。
- 竖向缩放条。
- 缩放百分比提示。
