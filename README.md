# 无限画布思维导图 V15.4

## 本版重点

### 1. 笔与手指分通道

- Apple Pencil / 触控笔继续使用 Pointer Events 书写。
- 在钢笔、水彩、激光和橡皮模式下，手指使用独立 Touch Events 平移画布。
- 手指平移不再要求退出绘图工具。
- Pointer Events 与 Touch Events 不再重复处理同一次手指输入。

### 2. 两个悬浮工具栏

- 原纯模式悬浮工具：负责钢笔、水彩、激光、橡皮、框和线。
- 新思维导图悬浮工具：负责移动、选择、新建节点、子节点、编辑、连接、更多操作和纯模式。
- 新工具栏可以拖动、折叠，并保存位置。
- 删除了底部固定菜单栏。

### 3. 书写稳定性

- 绘图模式下阻止网页文字选区、系统拖拽和长按菜单。
- 增加 window 级 pointerup / pointercancel 兜底。
- 指针捕获丢失时会完成当前笔画，不再让已写笔画消失。
- 浏览器导航栏、全屏状态或可视区域尺寸变化后自动恢复画布。

### 4. PDF

- PDF 可以按页码范围逐页展开到画布。
- PDF 页面作为画布对象，可移动和缩放。
- 笔迹直接画在 PDF 页面所在的画布坐标上。
- 默认建议一次导入 20 页，电子书可分批导入。
- 首次 PDF 渲染需要联网加载 PDF.js。

### 5. Word / PowerPoint

DOC、DOCX、PPT、PPTX 可以作为附件保存到笔记库并下载原文件。

纯 GitHub Pages 无法在不依赖转换服务的情况下可靠还原本地 Office 文件的原始版式，因此建议先转换为 PDF，再导入逐页批注。
