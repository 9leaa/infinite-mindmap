# 无限画布思维导图 V15

## 本版核心变化：IndexedDB 大容量存储

旧版本将整个笔记库存入 localStorage。手写笔迹、图片和富文本很容易触及几 MB 的 localStorage 上限。

V15 改为：

- 使用 IndexedDB 保存完整笔记库。
- 首次打开自动迁移 V14.1 及更早版本数据。
- 迁移成功后清理旧 localStorage 副本，释放空间。
- 保留完整笔记库备份和导入功能。
- 保留 GitHub Pages / PWA 离线访问。

## 升级前注意

旧版本若正在提示“浏览器存储空间不足”，不要立即关闭页面。

先点击：

`备份全部笔记`

该备份会读取当前页面内存中的最新笔记，即使最近一次自动保存失败，也可以导出当前内容。

然后打开 V15：

1. 如果旧数据仍在 localStorage，V15 会自动迁移。
2. 为确保没有遗漏，也可以导入刚刚导出的完整笔记库备份。

## 跨设备

IndexedDB 仍然属于当前设备和当前浏览器，不会自动跨设备同步。

换设备时：

1. 旧设备点击“备份全部笔记”。
2. 将 JSON 放到 iCloud Drive、Google Drive 等。
3. 新设备打开应用。
4. 点击“导入完整笔记库”。

真正自动同步仍需要账号系统和云数据库。
