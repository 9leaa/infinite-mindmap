# 无限画布思维导图 V15.6

## PDF 工作方式调整

导入 PDF 后，画布中只创建一个文件卡片，不再自动把每一页铺到无限画布。

点击文件卡片中的“打开阅读”后：

- 左侧继续显示思维导图和自由笔记。
- 右侧打开完整 PDF。
- PDF 采用连续滚动阅读。
- 页面按需渲染，不会一次把整本电子书全部绘制为高清 Canvas。
- 支持钢笔、荧光笔、整笔橡皮、缩放和阅读笔记。
- Apple Pencil 负责批注，手指可以继续滚动 PDF。
- 每页批注使用归一化坐标保存，缩放或重新打开后仍能恢复。
- 阅读笔记、批注和 PDF 原文件一同保存在当前笔记库。

## 性能说明

- PDF 页面使用 IntersectionObserver 延迟渲染。
- 只渲染当前阅读位置附近的页面。
- PDF Canvas 像素比最多为 2。
- 批注使用独立透明 Canvas。
- 放大、调整阅读区宽度时仅重新渲染当前附近页面。

## 存储说明

PDF 原文件会保存在 IndexedDB 中。大型电子书会明显增加：

- 浏览器存储占用
- 完整笔记库备份文件大小
- 首次导入时的内存占用

导入超过 60 MB 的 PDF 时，应用会再次确认。
