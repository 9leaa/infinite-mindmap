# 无限画布思维导图 V15.3

## 本版修复

- 纯模式工具按钮在手指或 Apple Pencil 按下时立即切换，不再依赖容易被取消的 click。
- 点击选择、钢笔、水彩、激光、橡皮、矩形或直线时，会自动退出平板“手掌移动”模式。
- 修复“图标显示已选择，但点击节点仍然在移动画布”的状态冲突。
- 平板纯模式按钮触控区域扩大到约 42 × 42 像素。
- 纯模式工具栏层级提高，不会被底部平板操作栏遮挡。
- 工具栏位置会自动限制在底部操作栏上方。
- 工具按钮增加 aria-pressed 状态和触摸按压反馈。

## 平板操作原则

- “移动”按钮开启时：一根手指从节点或空白处拖动画布。
- 点击任意绘图工具或“选择”后：自动退出移动模式。
- 选择工具下点击节点：选中节点。
- Apple Pencil 在钢笔、水彩、橡皮等模式下执行对应操作。
